# Hangman GUI Game 🎮

An interactive Hangman game built using Python and Tkinter. This version replaces traditional body parts with creative visual elements, making the gameplay unique and engaging. The project focuses on GUI design, user interaction, and logic building.

## Features
- Graphical User Interface (GUI)
- Sound effects for feedback
- Creative visual system (no body parts)
- Word guessing logic
- User-friendly gameplay

## Tech Stack
- Python
- Tkinter
- Playsound

## How to Run
1. Install Python
2. Run the following command:
   python main.py

## Project Structure
- main.py → Main game logic
- words.txt → Word list
- assets/ → Images and sound files
